Uberry Fancy Cake Shop 

Uberry Fancy Cake Shop is a responsive bakery website developed using HTML, CSS, and JavaScript. The website provides information about the cake shop, its products, services, and contact details through a modern and user-friendly interface.

Features
  Home Page
  Product Showcase
  Services Page
  Contact Page
  Responsive Design
  Smooth Navigation
  Modern User Interface

Technologies Used
  HTML5 
  CSS3
  JavaScript
